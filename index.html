    Battery Current Calculation

    Given:
    Battery voltage = 3.7 V
    Efficiency = 0.85

    I_batt = P_usb / (3.7 × 0.85)

    Mode 1:
    I_batt = 2.43 / 3.145 = 0.77 A

    Mode 2:
    I_batt = 9.29 / 3.145 = 2.96 A

    Battery Runtime

    Battery capacity = 0.5 Ah

    Mode 1:
    t = 0.5 / 0.77 = 0.65 hours ≈ 39 minutes

    Mode 2:
    t = 0.5 / 2.96 = 0.17 hours ≈ 10 minutes

    Logging Overhead

    ΔI = I_mode2 − I_mode1
    ΔI = 1.8 − 0.47 = 1.33 A

    ΔP = P_mode2 − P_mode1
    ΔP = 9.29 − 2.43 = 6.86 W

    Conclusion

    Mode 2 consumes more power due to SD card write operations.

    Logging significantly reduces battery runtime.
