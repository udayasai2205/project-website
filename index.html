    • Battery Runtime

    Battery capacity = 0.5 Ah

    Mode 1:
    t = 0.5 / 0.77 = 0.65 hours ≈ 39 minutes

    Mode 2:
    t = 0.5 / 2.96 = 0.17 hours ≈ 10 minutes

    Logging Overhead

    ΔI = Imode2 − Imode1
    ΔI = 1.8 − 0.47 = 1.33 A

    ΔP = Pmode2 − Pmode1
    ΔP = 9.29 − 2.43 = 6.86 W

    Conclusion

    Mode 2 consumes more power due to SD card write operations.

    Logging significantly reduces battery runtime.
